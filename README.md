## About Project

A calendar that displays all celestial events observable from the location of the user and recommends the optimal viewing time  based on weather data.

## Tech Stack

- Front End
  - HTML
  - CSS
  - JS
- APIs
  - Open Meteo API
  - AMS Restfull API
